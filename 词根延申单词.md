# 1.flex/flect 弯曲、转动

①reflect = re (back, 回/重新) + flect (bend, 弯曲)

- v. 反射（光线弯曲回来）；反省（思想转回自身）；反映（问题返回来）

②reflection = re (back, 回) + flect (bend, 弯曲) + ion (名词后缀)

- n. 反射；倒影；沉思/反省

③reflective = re (back, 回) + flect (bend, 弯曲) + ive (形容词后缀)

- adj. 反射的；沉思的；反光的

④deflect = de (away, 离开/偏离) + flect (bend, 弯曲)

- v. 使偏斜；使转向；使弯曲

⑤deflection = de (away, 离开) + flect (bend, 弯曲) + ion (名词后缀)

- n. 偏向；偏转；偏差

⑥inflect = in (towards, 向内) + flect (bend, 弯曲)

- v. 改变音调（声音弯曲）；使屈折（语法变形）

⑦inflection = in (towards, 向内) + flect (bend, 弯曲) + ion (名词后缀)

- n. 变调；屈折变化；语调的抑扬

⑧genuflect = genu (knee, 膝盖) + flect (bend, 弯曲)

- v. 屈膝；跪拜（膝盖弯曲表示尊敬）

⑨flex = flex (bend, 弯曲)

- v. 弯曲（肢体）；展示/炫耀（肌肉）；收缩

- n. 弹性织物

⑩flexible = flex (bend, 弯曲) + ible (able to be, 能...的)

- adj. 灵活的；易弯曲的；可变通的

⑪flexibility = flex (bend, 弯曲) + ibility (名词后缀)

- n. 灵活性；柔韧性；弹性

⑫inflexible = in (not, 不) + flex (bend, 弯曲) + ible (能...的)

- adj. 僵硬的；不可改变的；顽固的

⑬reflex = re (back, 回) + flex (bend, 弯曲)

- n. 反射作用（如膝跳反射）；本能反应

- adj. 反射的

⑭reflexive = re (back, 回) + flex (bend, 弯曲) + ive (形容词后缀)

- adj. 反身的（语法中的反身代词）；反射的

⑮circumflex = circum (around, 环绕) + flex (bend, 弯曲)

- n. 扬抑符（^ 这种弯曲的符号）

- adj. 弯曲的

⑯flexion = flex (bend, 弯曲) + ion (名词后缀)

- n. 弯曲；屈曲（解剖学术语，如关节弯曲）
